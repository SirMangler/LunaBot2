How to edit Settings

Existing Variables

todo - The todo list.
refix - The command prefix
afkmembers - Takes Tuples (value1:value2), indicates who (ID) is AFK and why. E.g 167445584142139394:because I am
announcementChannel - The channel that alerts are made in.
deleteDelay - How long temporary messages stay before being deleted.
liveAlert - Alert when kelsilynstar goes live or not.
followAlert - Alert when someone follows kelsilynstar.
customGreetings - Tuples, messages split by semi-colon, the custom greetings for individuals. 167445584142139394:Message 1;Message 2;Message 3;Et Cetera
debugMode - Unused currently.


Variables must start with %, then the data follows on the next line. A newline/blank line must follow at the end of the data, to indicate it's completion. See example.


Example file

 %todo
Eat cheese.

%afkmembers
167445584142139394:because I am

%prefix
*

%deletedelay
5

%customGreetings
167445584142139394:Message 1;Message 2;Message 3;Et Cetera

%liveAlert
true

%followAlert
true

%announcementChannel
442774379223449600
